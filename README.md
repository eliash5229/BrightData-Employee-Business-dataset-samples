Dieses **Bright Data Employee Business Dataset Sample** umfasst **2.519 Datensätze** und wurde mithilfe der **Bright Data API** extrahiert.

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Die URL des Datensatzeintrags. |
| `profile_url` | LinkedIn-Profil-URL der Person. |
| `linkedin_num_id` | Eindeutige numerische Kennung für das LinkedIn-Profil. |
| `avatar` | URL des Profilbilds, das mit dem LinkedIn-Konto verknüpft ist. |
| `profile_name` | Vollständiger Name der Person, wie in ihrem LinkedIn-Profil angegeben. |
| `certifications` | Details zu Zertifizierungen, die von der Person erworben wurden. |
| `profile_location` | Aktueller Standort der Person gemäß ihrem LinkedIn-Profil. |
| `profile_connections` | Ungefähre Anzahl der Kontakte, die die Person auf LinkedIn hat. |
| `profile_country_code` | ISO-Code, der das Land des Standorts der Person darstellt. |
| `profile_education_full` | Detaillierte Bildungshistorie der Person. |
| `profile_last_education` | Aktuellster Bildungsabschluss der Person. |
| `profile_experience_full` | Umfassende Details zur Berufserfahrung der Person. |
| `profile_last_experience` | Aktuellste Position und Arbeitgeber. |
| `profile_followers` | Anzahl der Follower, die die Person auf LinkedIn hat. |
| `profile_linkedin_id` | LinkedIn-ID-String der Person. |
| `profile_current_position` | Aktuelle berufliche Position der Person. |
| `profile_current_title` | Aktueller Jobtitel der Person. |
| `profile_activity` | Aktuelle Aktivitäten, die die Person auf LinkedIn durchgeführt hat. |
| `profile_posts` | Beiträge, die von der Person auf LinkedIn veröffentlicht wurden. |
| `profile_about` | Zusammenfassungsbereich des LinkedIn-Profils der Person. |
| `profile_courses` | Kurse, die von der Person absolviert wurden. |
| `profile_volunteer_experience` | Ehrenamtliche Arbeitserfahrung der Person. |
| `profile_languages` | Sprachen, die von der Person gesprochen werden. |
| `profile_publications` | Veröffentlichungen, die von der Person verfasst wurden. |
| `profile_recommendations` | Empfehlungen, die von der Person erhalten oder gegeben wurden. |
| `profile_recommendations_count` | Gesamtzahl der Empfehlungen, die mit der Person verbunden sind. |
| `profile_organizations` | Organisationen, denen die Person angehört oder angehörte. |
| `profile_projects` | Projekte, die von der Person durchgeführt wurden. |
| `company_name` | Name des Unternehmens, bei dem die Person derzeit beschäftigt ist oder zuvor beschäftigt war. |
| `company_id` | Eindeutige Kennung für das Unternehmen innerhalb des Datensatzes. |
| `company_linkedin_url` | LinkedIn-URL des Unternehmens. |
| `company_size` | Größe des Unternehmens anhand der Mitarbeiterzahl. |
| `company_country_code` | ISO-Code, der das Land des Unternehmenssitzes darstellt. |
| `company_description` | Beschreibung der Geschäftstätigkeiten des Unternehmens. |
| `company_other_employees` | Anzahl weiterer Mitarbeiter, die auf LinkedIn aus demselben Unternehmen aufgeführt sind. |
| `employees_in_linkedin` | Gesamtzahl der Mitarbeiter des Unternehmens, die auf LinkedIn aufgeführt sind. |
| `company_linkedin_followers` | Anzahl der Follower, die das Unternehmen auf LinkedIn hat. |
| `company_locations` | Liste der Unternehmensstandorte. |
| `company_founded_year` | Gründungsjahr des Unternehmens. |
| `company_headquarters` | Standort des Hauptsitzes des Unternehmens. |
| `company_categories` | Branchen-Kategorien, in denen das Unternehmen tätig ist. |
| `company_logo` | URL des Unternehmenslogos. |
| `company_slogan` | Slogan oder Tagline, die vom Unternehmen verwendet wird. |
| `company_specialties` | Spezialisierungen und Fachgebiete des Unternehmens. |
| `company_updates` | Aktuelle Updates oder Neuigkeiten über das Unternehmen. |
| `company_website` | Offizielle Website des Unternehmens. |
| `company_type` | Unternehmensart, z. B. börsennotiert, privat, Non-Profit. |
| `company_clean_domain` | Bereinigter Domainname der Unternehmenswebsite. |
| `company_revenue_usd` | Ungefähre Einnahmen des Unternehmens in USD. |
| `company_total_funding` | Gesamte Finanzierung, die das Unternehmen erhalten hat. |
| `company_total_employees` | Gesamtzahl der Mitarbeiter, die im Unternehmen arbeiten. |
| `company_stock_symbol` | Börsensymbol des Unternehmens, falls es öffentlich gehandelt wird. |
| `company_is_non_profit` | Indikator, ob das Unternehmen eine Non-Profit-Organisation ist. |
| `company_parent_company` | Name der Muttergesellschaft, falls zutreffend. |
| `profile_bio_links` | Links, die im Biografieabschnitt des Profils der Person enthalten sind. |


## Use Cases

### 1. Human Resources Analytics
Verbessern Sie HR-Strategien, indem Sie Bildung, Erfahrung und Zertifizierungen von Mitarbeitern analysieren, um Kompetenzlücken sowie Schulungsbedarfe zu identifizieren.

### 2. Market Research
Führen Sie Marktforschung durch, indem Sie Mitarbeiterprofile branchenübergreifend untersuchen und Unternehmen dabei helfen, Wettbewerbsdynamiken sowie die Verteilung von Belegschaften zu verstehen.

### 3. Business Development
Nutzen Sie umfassende Mitarbeiter- und Unternehmensdaten, um Partnerschaftsmöglichkeiten aufzudecken und Maßnahmen zur Geschäftsexpansion zu strategisieren.
